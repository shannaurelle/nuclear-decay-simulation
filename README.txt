Nuclear Decay Simulation 
version 1.0
Created by Shann Aurelle Ripalda, Michelle Healmer Castro, and Natalie Shayne Macababbad
to run program, please click on mainSimulation.exe
thanks for downloading!

To edit the number of particles and
the temperature (in Kelvin) open
data.txt. The first value is the number of 
particles while the second value after 
the comma is the temperature, which determines
the speed of the simulation particles
format : numberOfParticles,temperature

Runs on Windows, going to test on MacOS and Linux soon
